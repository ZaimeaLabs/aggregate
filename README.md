## Introduction

``Aggregate`` is a Laravel package that allows you to generate aggregates from your Eloquent models or queries. It makes it easy to calculate totals, averages, sums, counts, and cumulative values over customizable time intervals. Designed to simplify reporting and chart generation, Aggregate helps you quickly analyze data trends and produce meaningful insights from your application's data.
